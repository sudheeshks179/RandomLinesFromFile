# RandomLinesFromFile
Print n random lines from file.<br />
## Compilation:<br />
   go build<br />

## Run<br />
./RandomLinesFromFile -filename /tmp/file.txt -numberOfLines 100 -avgLineSize 10 <br />
or<br />
./RandomLinesFromFile -filename words_alpha.txt -numberOfLines 100<br />
or<br />
./RandomLinesFromFile noLinestoPrint <br />
eg: ./RandomLinesFromFile 4
